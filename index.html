HTML & CSS Basics summary

HTML summary

(HYPERTEXT MARKUP LANGUAGE) is the standard markup language for docments designed to be displayed in a web browser. it defines the meaningspan and structure of web content. it is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

  • HTML uses "element to describe (annotate content)
  • HTML element typically have an opening tag, content and then a closing
  • You can also have void (empty) elements like images
  • You can also configure element eith attributes
  • There's is a long list of available elements but you'll gain experience over time, no worries.

Learn more about all available HTML elements on the MDN HTML element reference.

CSS summary

CSS (CASCADING STYLE SHEETS) is a style sheet language used for describing the presentstion of a document written in a markup language such as HTML or XML.CSS is a cornerstone technology of the world widee web, alongside HTML and JAVASCRIPT. CSS is a computer language for laying out and stucting web pages (HTML or XML). This language contains coding elements and it compose of these "cascading style sheets" which are equally called CSS files (.

  • Styles are assigned vis property-value pairs
  • You can assign styles via the "attribute"
  • To avoid code duplication, you typcally use global styles( via the "style" element) instead
  • Alternatively, you can work with external stylesheet files which you "link"to
  • when working with CSS, concepts like "inheritance", "specificity" and the "box model" should be understood
